# Fuzzy Logic & ANFIS Project

## 📌 Overview

This project contains two parts:

1. **Fuzzy Logic System for Washing Machine**
2. **Hybrid Intelligent System (ANFIS) for Student Performance Prediction**

---

## 🧠 Q1: Washing Machine Fuzzy System

### Inputs:

* Dirt Level (Low, Medium, High)
* Load Size (Small, Medium, Large)

### Output:

* Cycle Time (Short, Medium, Long)

### Description:

A fuzzy logic system is designed using MATLAB to determine the washing machine cycle time based on dirt level and load size.

---

## 🤖 Q2: Student Performance Prediction (ANFIS)

### Inputs:

* Attendance
* Assignment Marks
* Test Marks

### Output:

* Performance Level (Poor, Average, Good)

### Description:

A hybrid system combining fuzzy logic and neural networks (ANFIS) is used to predict student performance.

---
